# 文件存储服务
> 基于Koa2开发的文件存储服务

## 开发
### 本地开发运行
```
npm run dev
```
- 前端测试页面文件夹 views
    - 小文件测试页面：http://localhost:3000/
    - 大文件测试页面：http://localhost:3000/fenpian
